Yöntemsel Erişim

Nesnelere erişim sağlayabilen yöntemlerden biride yöntemsel çağrıdır.

Performans açısından en yavaş çağrıdır. Gerek kalmadıkça kullanmaktan kaçınmanız önerilir.

Controllers/Anasayfa.php 
<?php
class Anasayfa
{
	function index()
        {
            this()->import->page('Anasayfa');
        }
}

Böyle bir kullanım için herhangi bir yapıyı extends etmeye gerek yoktur.

this()->import->library('Session');

this()->sess->insert('kullanici', 'zntr');

echo this()->sess->select('kullanici');

Her sayfa üzerinden nesnelere erişim imkanı sağlar.
