Basit Bir Uygulama Yapmak

Aşağıda uygulamanın aşamaları yer almaktadır.

1.Adım: Views/Pages/ dizinine iskelet.php ve anasayfa.php adında 2 sayfa oluşturun.
2.Adım: Views/Pages/iskelet.php sayfasının içerisine şu kodları ilave edin.

Views/Pages/iskelet.php 
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title><?php echo $baslik; ?></title>
</head>

<body>
	<?php echo $ic_sayfa; ?>
</body>
</html>

3.Adım: Views/Pages/anasayfa.php sayfasının içerisine "Merhaba ben anasayfayım" yazısını ilave edin.
4.Adım: Controllers/ dizinine anasayfa.php adında bir sayfa oluşturun.
5.Adım: Controllers/anasayfa.php sayfasının içerisine şu kodları ilave edin.
Uyarı: Controllers/anasayfa.php dosyasının adı anasayfa olduğundan dolayı bu sayfanın sınıf adıda anasayfa olmalıdır.

Controllers/Anasayfa.php 
<?php
class Anasayfa{

    function index($params = "")
    {
    	$veriler["baslik"] = "Anasayfa"; // Çağrılan sayfaya başlık veriyoruz.
    
        $veriler["ic_sayfa"] = import::page("anasayfa", "", true); // Views/Pages/anasayfa.php sayfasının içeriğini dizi değişkene aktarıyor.
        
        import::page("iskelet", $veriler, false); // Views/Pages/iskelet.php sayfasını çağrıyor.
    }
}

Şimdi URL Adresinizi Çalıştırın

Not:Eğer URL'niz parametre içermiyorsa ve fonksiyonunuzun adı index ise yazmanıza gerek yoktur.

Localhostta çalışıyorsanız adres çubuğuna:localhost/index.php/anasayfa/index yazın.

Sunucuda çalışıyorsanız adres çubuğuna:www.siteadi.xxx/index.php/anasayfa yazın. Gerekli olmadığı için bu URL adresinde index ibaresini kullanmadık.
