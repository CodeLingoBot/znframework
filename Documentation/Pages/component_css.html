Animation bileşenine ait yöntemler aşağıdaki tabloda listelenmiştir

Animation Bileşenini Dahil Etmek

import::component('Css/Animation')

Animasyon Bileşeni ve YöntemleriAnlamlarıKullanımları
selector( $erisim )Nesneye erişim sağlamak için kullanılan adlandırmalar. Örn: .sinif, #id ...$this->animation->selector('Erişim Adı')
attr( $ozellik_deger_dizisi )Animasyon öğeleri dışında eklenecek özellikler ve değerleri belirtir. Bu yöntem dizi parametresi içerir.->attr(array('border' => 'solid 1px #000'))
name( $isim )animation-name: parametresinin karşılığıdır.->name('animation')
direction( $yon = 'reverse' )animation-direction: parametresinin karşılığıdır.->direction('reverse')
status( $durum )animation-play-state: parametresinin karşılığıdır.->status('pause')
fill( $mod )animation-fill-mode: parametresinin karşılığıdır.->fill('forwards')
repeat( $tekrar )animation-iteration-count: parametresinin karşılığıdır.->repeat(3)
duration( $sure )animation-duration: parametresinin karşılığıdır.->duration(.5)
delay( $baslama_suresi )animation-delay: parametresinin karşılığıdır.->delay(.5)
easing( $etki_turu )animation-timing-function: parametresinin karşılığıdır.->easing('ease-in-out')
complete()Çoklu animasyon oluşturmak için kullanılan tamamlama yöntemidir.->complete();
create()Animasyonu tamamlama yöntemidir.->create();

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

$this->animation
     ->selector('#nesne')
     ->direction('reverse')
     ->repeat(3)
     ->easing('ease-in-out')
     ->create();

Çoklu animasyon oluşturmak için aşağıdaki gibi bir kullanım gerekir.

$kuyruk = $this->animation
     ->direction('reverse')
     ->repeat(1)
     ->easing('ease-out-bounce')
     ->complete();
      
$this->animation
     ->name('animation')
     ->selector('#nesne')
     ->direction('reverse')
     ->repeat(3)
     ->easing('ease-in-out')
     ->create($kuyruk);

$this->transition

Shadow Bileşeni Yöntemleri

Shadow bileşenine ait yöntemler aşağıdaki tabloda listelenmiştir

Shadow Bileşenini Dahil Etmek

import::component('Css/Shadow')

Shadow Bileşeni ve YöntemleriAnlamlarıKullanımları
selector( $erisim )Nesneye erişim sağlamak için kullanılan adlandırmalar. Örn: .sinif, #id ...$this->shadow->selector('Erişim Adı')
attr( $ozellik_deger_dizisi )Animasyon öğeleri dışında eklenecek özellikler ve değerleri belirtir. Bu yöntem dizi parametresi içerir.->attr(array('border' => 'solid 1px #000'))
x( $x )box-shadow: yönteminin 1. parametresinin karşılığıdır.->x(1)
horizontal( $x )box-shadow: yönteminin 1. parametresinin karşılığıdır. x() yönteminin alternatifidir.->horizontal(20)
y( $y )box-shadow: yönteminin 2. parametresinin karşılığıdır.->y(3)
vertical( $y )box-shadow: yönteminin 2. parametresinin karşılığıdır. y() yönteminin alternatifidir.->vertical(2)
blur( $gorunurluk )box-shadow: yönteminin 3. parametresinin karşılığıdır.->blur(2)
diffusion( $yayilma )box-shadow: yönteminin 4. parametresinin karşılığıdır.->diffusion(10)
spread( $yayilma )box-shadow: yönteminin 4. parametresinin karşılığıdır. diffusion() yönteminin alternatifidir.->spread(20)
color( $renk )box-shadow: yönteminin 5. parametresinin karşılığıdır. Gölgenin renk bilgisini içerir.->color('#000')
create()Gölge işlemini tamamlama yöntemidir.->create();

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

$this->shadow
     ->selector('#nesne')
     ->x(10) // ya da horizontal()
     ->y(20) // ya da vertical()
     ->blur(3)
     ->diffusion(3) // ya da spread()
     ->color('red')
     ->create();

close()Style tagı kapatma işlevini üstlenir.->close();

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

$this->style->open()
	// stil kodları 
$this->style->close()

Style tagı açarken farklı harici css dosyası dahil etmek isterseniz.

$this->style->library('style')->open()
	// stil kodları 
$this->style->close()
