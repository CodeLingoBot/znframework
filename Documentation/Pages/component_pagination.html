import::component('Pagination')

Sayfalama Bileşeni Yöntemleri

Sayfalama bileşenine ait yöntemler aşağıdaki tabloda listelenmiştir

Sayfalama Oluşturma YöntemleriAnlamlarıKullanımları
url( $url )Sayfalamanın kullanılacağı URL adresi.$this->pag->url('http://www.ornek.xxx/urunler/sayfa/')
start( $baslangic )Kaçıncı kayıttan listelenmeye başlayacağı.->start(0)
limit( $limit )Tek bir sayfada olması istenilen kayıt sayısı.->limit(10)
total_rows( $toplam_kayit )Toplam kayıt sayısı.->total_rows(200)
count_links( $link_sayisi )Sayfalamada yer alacak link sayısı.->count_links(10)
attr( $ozellikler )Limit, total_rows gibi sayfalama özelliklerinin hepsini dizi tipinde bu yöntem ile belirtebilirsiniz.->attr(array('start' => 0, 'limit' => 10))
style( $style )Sayfalama nesnelerine stil eklemek için kullanılır.->style(array('current' => 'color:red'))
css( $css )Sayfalama nesnelerine css sinifi eklemek için kullanılır.->css(array('current' => 'red-color'))
create( [ $start ], [ $start ], [ $total_rows ], [ $url ] )Sayfalama oluşturma yöntemidir. İsteğe bağlı 4 parametresi vardır.->create();

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

$this->pag->url('urunler/sayfa')->limit(10)->total_rows(200)->start(0)->create(); 

$this->pag
     ->url('urunler/sayfa')
     ->limit(10)
     ->total_rows(200)
     ->start(0)
     ->style(array('current' => 'color:blue', 'font-size' => '11px'))
     ->create();
