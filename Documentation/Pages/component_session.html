Oturum Bileşeni Yöntemleri

Oturum bileşenine ait yöntemler aşağıdaki tabloda listelenmiştir

Oturum Oluşturma YöntemleriAnlamlarıKullanımları
name( $ad )Oturum ismi.$this->sess->name('Oturum Adı')
value( $deger )Oturum değeri.->value('Değer')
encode( $anahtar_hash_algo , $deger_hash_algo )Oturumu şifrelemek için kullanılır. 2 parametresi vardır. 1. parametre çerezin anahtar değerini, 2. parametre oturumun tuttuğu veriyi şifrelemek için kullanılır. Bu yöntemin kullanımı isteğe bağlıdır. Sadece anahtar veya sadece değer bilgisini şifreleyebilirsiniz.->encode('md5', 'sha1')
create( [ $name ] , [ $value ] )Oturum oluşumunu tamamlama yöntemidir. Ancak isteğe bağlı olarak isim veya değer bilgisi içerebilir.->create('oturum', 'Değer'); // Sonlandırma komutu

Yukarıdaki yöntemlerin kullanımına yönelik örnekler aşağıda verilmiştir.

Kısayoldan oturum oluşturmak için;

$this->sess->create('ornek_isim', 'Örnek Değer');

İsim ve değer için yöntem kullanma;

$this->sess->name('ornek_isim')->value('Örnek Değer')->create();

Oturumu şifrelemek;

$this->sess->encode('sha1')->create('ornek_isim', 'Örnek Değer');

Oturum Kullanma YöntemleriAnlamlarıKullanımları
name( $ad )Oturum ismi.$this->sess->name('oturum')
decode( $anahtar_hash_algo )Anahtarı şifrelenen oturumu kullanabilmek için kullanılır. Hangi algoritma ile şifrelenmişse o algoritma ile kullanılmak durumundadır.->decode('md5')
select( [ $name ] )Oturum seçimini tamamlama yöntemidir. Ancak isteğe bağlı olarak isim bilgisi içerebilir.->select('oturum');

Kısayoldan oturumu kullanmak için;

$this->sess->select('ornek_isim'); // Örnek Değer

Şifrelenmiş oturumu kullanmak için;

$this->sess->decode('sha1')->select('ornek_isim'); // Örnek Değer

Tüm oluşturulumuş oturumları listelemek;

$this->sess->select(); // Parametre belirtilmediği taktirde daha önce oluşturulmuş tüm oturumların listesini verir.

delete( [ $name ] )Oturum silme işlemini tamamlama yöntemidir. Ancak isteğe bağlı olarak isim bilgisi içerebilir.->delete('oturum');

Kısayoldan oturumu silmek için;

$this->sess->delete('oturum');

Şifrelenmiş oturumu silmek için;

$this->sess->decode('sha1')->delete('oturum');

Tüm oluşturulumuş oturumu silmek için;

$this->sess->delete(); // Parametre belirtilmediği taktirde daha önce oluşturulmuş tüm oturumların silinmesini sağlar.
