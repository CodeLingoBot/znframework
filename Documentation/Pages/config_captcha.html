Captcha(Güvenlik Kodu) Ayarları

Güvenlik kodunun özellikleri ile ilgili bir takım ayarlamaların yapıldığı dosyadır. Genellikle güvenlik kodunun görünüşü ile ilgili ayarları içerir.

Config/Captcha.php 
/* SETTINGS	*/
// İşlev:Güvenlik kodu görselinin ayarlarını yapmak için kullanılır
// Parametre:Anahtar değer çifti içeren bir dizi ayar bilgisi içerir
// Örnek: array('char_count' => '6');

$config['Captcha']['settings'] = array
(
    //Güvenlik kodu resminin kaç karakterden oluşsacağı = 0-12
    "char_count"	=> "6", 
    //Güvenlik kod resminin arkplan resmi ve resimleri tek resim kullanılacaksa string değerde girilebilir. 
    //Örnek:array('resimler/bg1.jpg', ''resimler/bg2.jpg',')
    "background"	=> array(), 
    //Güvenlik kod resminin arkaplan rengi = 0-255|0-255|0-255
    "bg_color"		=> "0|0|0", 
    //Güvenlik kodu yazısının rengi = 0-255|0-255|0-255
    "font_color" 	=> "255|255|255", 
    //Güvenik kodu resminin çevçevesi olsun mu olmasın mı? = true-false
    "border" 		=> true, 
    //Güvenlik kodu resminin çerçeve renk kodu = 0-255|0-255|0-255
    "border_color" 	=> "0|0|0", 
    //Güvenlik kodu resminin genişliği = 0-9999
    "width"		=> "125",
    //Güvenlik kodu resminin yükseklik = 0-9999
    "height" 		=> "40", 
    //Yazının Boyutu, Yataydaki Konumu, Dikeydeki Konumu
    "image_string" 	=> array("size" => "5", "x" => "35", "y" => "13"), 
    //Grid olsun mu olmasın mı? = true-false
    "grid"		=> false, 
    //Gridler arası boşluğun yatayda veya dikeyde kaç kare olacağını ayarlar. 
    //Varsayılan olarak yatayda 12 ve dikeyde 4 kareden oluşan toplam 48 gridden oluşacaktır.
    "grid_space"	=> array("x" => 12 , "y" => 4), 
    //Gridin renk kodu = 0-255|0-255|0-255
    "grid_color" 	=> "0|0|0" 
);

Genel olarak güvenlik kodu ayarları yukarıdaki gibidir siz istediğiniz gibi ayarlayabilirsiniz.
