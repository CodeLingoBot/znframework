Sayfa Çağırma

Kod Çatısında sayfa çağırmak için import::page("sayfa adı") veya import::view("sayfa adı") kodu kullanılır. Bu komut Import sınıfını anlatırken daha detaylı olarak incelenecektir.

Parametreler

import::page() komutu 3 parametreden oluşur. Bunlar; import::page(sayfa adı, [sayfaya gönderilecek veriler], [sayfanın içeriği]); parametleridir.

1 - Sayfa Adı:Views/Pages dizini içerisinden çağrılacak herhangi bir sayfanın adıdır. Eğer çağrılacak sayfa .php uzantılı ise uzantının kullanımına gerek yoktur.
2 - Gönderilecek Veriler:Çağrılan sayfanın içerisine veri gödermek için kullanılır. Veriler dizi değişken türünde gönderilir.
3 - Sayfanın İçeriği:Çağrılan sayfa içeriğinin tamponlanarak başka bir nesneye aktarılmasında kullanılır. Parametre true veya false değeri alır varsayılan değer false'dir. Şayet bu değer true olarak ayarlanırsa içerik metinsel(string) veri halini alır ki böyle bir kullanımın amacı sayfanın içeriğini veri olarak alıp başka bir sayfaya göndermektir.

Aşağıda bu anlatıma uygun örnek bir kod yer almaktadır.

Controllers/home.php

<?php
class Home{
    function index($params = "")
    {
        $veriler['title'] = "Anasayfa";
        // Views/Pages/ic_sayfa.php sayfasının içeriği bir dizi değişkene veri olarak aktarılıyor.
        $veriler['ic_sayfa'] = import::page("ic_sayfa", "", true);
        
        import::page("anasayfa", $veriler);
    }
}

Yukarıdaki kullanımda dikkat etmişseniz bir sayfa, başka bir sayfaya veri olarak gönderiliyor.

Eğer çağırdığımız sayfanın içerisinde $title değişkeni kullanmışsak bu değişken "Anasayfa" değerini almış olur. Yani dizi içinde gönderdiğimiz verilerin anahtar ifadeleri, çağrılan sayfanın içerisinde değişken olarak kullanılır.

Örnek: Views/Pages/anasayfa.php dosyası içerisinde şöyle bir ifade olsun;

Views/Pages/anasayfa.php

<?php

echo $title; // Çıktı: Anasayfa
echo $ic_sayfa; // Çıktı: İç sayfanın içeriği.

Adres çubuğuna gireceğimiz URL adresi www.siteadi.xxx/index.php/home/index'dir. Eğer index.php için gerekli ayarlar yapılmışsa URL adresi www.siteadi.xxx/home/index olur.

Gördüğünüz gibi Controllers/ dizininden çağrılan sayfa içerisine dizi olarak gönderilen anahtar ifadeler, çağrılan sayfa içerisinde değişken olarak kullanılıyor.
