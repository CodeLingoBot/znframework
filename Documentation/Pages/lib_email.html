Email Kütüphanesini Dahil Etmek

import::library('Email');

Genel E-posta Ayarlarını Yapılandırmak

Genel e-posta ayarlarını yapılandırmak için Config/Email.php dosyasını açın. Email.php dosyası açıldığında aşağıdaki ayarlarla karşılacaksınız. Bu dosya üzerinden ayarlamaların yapılması sizi tekrar tekrar aynı ayarların yazılması yükünden kurtaracaktır. Önemli olan ayarlar koyu olarak gösterilmiştir.

$config["Email"] = array(
	'username' 			=> '',	// Gerçek bir mail adresi. Örn: bilgi@zntr.net string kullanıcı adı
	'fromname' 			=> '',	// Gönderici ismi. Örn: Zntr string gönderici ismi
	'password' 			=> '',	// Gerçek mail adresinin şifresi. Örn:zntr1234 string şifre
    	'host' 				=> '',	// Mail sunucusunun adı. Örn: mail.zntr.net string host adı
	'port' 				=> 587,	// Bağlantı sağlanacak port numarası. int numara
	'is_html' 			=> true, // Html içerikli mail gönderilsin mi? boolean true veya false
	'is_smtp' 			=> true, // Smtp kullanılsın mı? boolean true veya false
	'smtp_auth'	 		=> true, // Kimlik doğrulansın mı? boolean true veya false
	'smtp_debug' 			=> true, // Smtp hata ayıklama açık mı? boolean true veya false
	'smtp_secure' 			=> 'tsl', // Bağlantı öneki. Alabileceği değerler string "", "ssl" veya "tls"
	'smtp_keep_alive'		=> false, // Değerin true Her mail gönderiminde smtp bağlantısının kapatılmasını önler. boolean true veya false
	'charset' 			=> 'UTF-8', // İçerik karakter kodlaması. string karakterseti
	'alt_body'			=> '', // Normal mail gönderilemediği durumda alternatif içerik göndermek için kullanılır. string alternati içerik
	'priorty' 			=> '', // Mailin öncelik derecesi. Varsayılan değer: 3 Alabileceği değerler: int 1 = Yüksek, 3 = Normal, 5 = Düşük
	'content' 			=> '', // Mailin içeriğinin tipi. Varsayılan değer: string 'text/plain'
	'encoding'			=> '', // Mesajı kodlama ayarları. Varsayılan değer: 8bit. Alabileceği değerler: string "8bit", "7bit", "binary", "base64" ve "quoted-printable".
	'word_wrap'			=> '',	// Mesajın kaç satırlık aşağı kayacağını ayarlar. Varsayılan değer: 0. 
	'send_mail' 			=> '', // Mail gönderme programının yolunu ayarlar. Varsayılan değer: "/usr/sbin/sendmail"
	'mailer' 			=> '', // Mail gönderme yöntemini belirler. Varsayılan değer: 'mail'. Alabileceği değerler: "mail", "sendmail", "smtp"
	'sender' 			=> '', // return_path parametresi boş değil ise maili smtp modunda gönderir.
	'return_path' 			=> '', // Bu parametre boş olursa sender parametresi ayarlanmalıdır.
	'use_send_mail_options' 	=> '', // Mail gönderme seçenekleri ayarlanabilsin mi? Varsayılan: true. Alabileceği değerler: true veya false
	'confirm_reading_to' 		=> '', // Okuma onayı gönderilecek mail adresi ayarlanır.
	'host_name' 			=> '', // Bu değer boş olursa bu ayarı ana bilgisayar ayarlar.
	'message_id' 			=> '', // Mesaj kimliği belirlenir.
	'message_date' 			=> '', // Mesaj tarihini ayarlar.
	'helo' 				=> '', // İletiyi SMTP HELO olarak ayarlar. Varsayılan: hostname parametresidir.
	'auth_type' 			=> '', // SMTP Auth türün ayarlar. Varsayılan: LOGIN. Alabileceği değerler: string LOGIN, PLAIN, NTLM
	'plugin_dir' 			=> '', // Eklentilerin yolunu ayarlar. string yol
	'realm'	 			=> '', // SMTP Bölge ayarı. string bolge
	'work_station' 			=> '', // SMTP Çalışma istasyonu ayarı. string bolge
	'timeout' 			=> '', // Saniye cinsinden zaman aşımı süresi. Varsayılan değer:10. Alabileceği değer: int saniye
	'debug_output' 			=> '', // Hata ayıklama çıktısının yazdırılma yöntemini ayarlar. Varsayılan değer: 'echo'. Alabileceği değerler: string "echo" veya "error_log"
	'single_to' 			=> '', // Bireysel mail gönderebilme imkanı sağlar. Varsayılan değer: false. Alabileceği değerler: boolean true veya false.
	'single_to_array' 		=> array(), // single_to parametresi true ise mail adreslerinin bir dizide tutar.
	'le' 				=> '', // Bir alt satıra geçmek için "\n" ibaresi kullanır. Varsayılan değer: \n.
	'dkim_selector'			=> '', // DKIM Yapılandırmaları.
	'dkim_identity'			=> '', // DKIM Yapılandırmaları.
	'dkim_pass_phrase' 		=> '', // DKIM Yapılandırmaları.
	'dkim_domain' 			=> '', // DKIM Yapılandırmaları.
	'dkim_private' 			=> '', // DKIM Yapılandırmaları.
	'action_function' 		=> '', // Geri dönüş yöntemini ayarlar.
	'version' 			=> '',	// Versiyon bilgisini içerir.
	'xmailer' 			=> '' // X-Mailer başığında ne kullanılacak. Varsayılan değer:NULL
);
Bir e-posta gönderebilmek için öncelikli olarak yapılandırılması gereken ayarlar.
--------------------------------------------------------------------------------------------------------------------------------
1-'username' => '', // Gerçek bir mail adresi. Örn: bilgi@zntr.net string kullanıcı adı
2-'password' => '', // Gerçek mail adresinin şifresi. Örn:zntr1234 string şifre
3-'host' => '', // Mail sunucusunun adı. Örn: mail.zntr.net string host adı
4-'port' => 587, // Bağlantı sağlanacak port numarası. int numara

NOT
Değer verilmeyen ayarlar sınıfın varsayılan değerini alacaktır.

Dahili E-posta Ayarlarını Yapılandırmak

email::settings( array $ayarlar )

E-posta göndermek için ayarların yapılandırılmasında kullanılan bu yöntemin tek bir dizi parametresi vardır. Ayarlar.

Eğer bir ayar hem Config/Email.php dosyasında hemde yöntem kullanılarak yapılırsa yöntemde kullanılan ayarın değeri kullanılır.

ParametreAnlamları
1. Parametre = AyarlarConfig/Email.php içerisinde bulunan her ayarı kullanabilirsiniz.

import::library('Email');

$ayarlar = array(
	'host' => 'mail.zntr.net',
    	'port' => 587,
    	'username' => 'bilgi@zntr.net',
    	'password' => 'zntr1234'
);

email::open();
email::settings($ayarlar);

NOT
Config/Email.php dosyası üzerinden yapılan her ayarı bu yöntemlede yapabilirsiniz ancak aynı ayarı iki yerdede yapmışsanız son olarak yöntemde yapıldığı için yöntemin ayarları dikkate alınacaktır.

E-posta Alıcısı Eklemek

email::receiver( string $eposta , string $isim )

E-posta alıcısı eklemek için kullanılır 2 parametresi vardır. Gönderilecek E-posta, Herhangi Bir İsim.

ParametreAnlamları
1. Parametre = Alıcı E-posta AdresiGönderilecek alıcının e-posta adresi.
2. Parametre = [Herhangi Bir İsim]Göndericinin adı veya alıcının adı yazılabileceği gibi herhangi bir konu ismide yazılabilir.

import::library('Email');

$ayarlar = array(
	'host' => 'mail.zntr.net',
    	'port' => 587,
    	'username' => 'bilgi@zntr.net',
    	'password' => 'zntr1234'
);

email::open();
email::settings($ayarlar);
email::receiver('iletisim@zntr.net','zntr');
email::close();

Birden fazla alıcı eklemek istersek 2 yöntem kullanabilirsiniz.

email::receiver('iletisim@zntr.net','zntr');
email::receiver('admin@zntr.net','zntr');

Veya

email::receiver(array('iletisim@zntr.net' => 'zntr', 'admin@zntr.net' => 'bilgi'));

E-posta Göndericisi Eklemek

email::sender( string $eposta , string $isim )

E-posta alıcısı eklemek için kullanılır 2 parametresi vardır. Göndere E-posta Adresi, Gönderen Kişinin İsmi.

Bilmeniz gereken nokta şayet bir gönderici belirlemezseniz varsayılan olarak ayarlardaki username parametresinin değerini alacaktır. Bu sebeple kullanımı zorunlu değildir.

ParametreAnlamları
1. Parametre = Gönderici E-posta AdresiGönderilecek alıcının e-posta adresi.
2. Parametre = [Herhangi Bir İsim]Göndericinin adı veya alıcının adı yazılabileceği gibi herhangi bir konu ismide yazılabilir.

import::library('Email');

$ayarlar = array(
	'host' => 'mail.zntr.net',
    	'port' => 587,
    	'username' => 'bilgi@zntr.net',
    	'password' => 'zntr1234'
);

email::open();
email::settings($ayarlar);
email::sender('bilgi@zntr.net','zntr');
email::receiver('iletisim@zntr.net','zntr');
email::close();

E-posta Göndermek

email::send( [ string $konu ] , [ string $mesaj ] )

Oluşturulan e-postayı göndermek için kullanılan yöntemdir. 2 isteğe bağlı parametresi vardır. Konu, Mesaj.

ParametreAnlamları
1. Parametre = [Mesaj]E-postanın içeriği.
2. Parametre = [Konu]E-postanın içeriği hakkında bilgi veren başlık bilgisi.

import::library('Email');

$ayarlar = array(
	'host' => 'mail.zntr.net',
    	'port' => 587,
    	'username' => 'bilgi@zntr.net',
    	'password' => 'zntr1234'
);

email::open();
email::settings($ayarlar);
email::sender('bilgi@zntr.net','zntr');
email::receiver('iletisim@zntr.net','zntr');
email::subject('Ornek Mail Gönderimi');
email::message('Merhaba ben zntr örnek bir mail gönderdim.');
email::send();  // Zaten konu ve mesaj bilgisini oluşturduğumuzdan dolayı parametrelerini kullanmadık.
email::close();

Daha pratik e-posta göndermek için kodları şu şekilde düzenliyoruz.

import::library('Email');

 // Ayarları Config/Email.php dosyasından yapınız.

email::open();
email::receiver('iletisim@zntr.net','zntr');
email::send('Konu', 'Mesaj'); 
email::close();

E-posta İçeriğine Dosya Eklemek

email::add_attachment( string $dosya , string $isim , [ string $sifreleme = 'base64' ] , [ string $tip = 'application/octet-stream' ] )

Gönderilecek e-posta içerisinde dosya göndermek için kullanılır. 4 parametresi vardır. Dosya, İsim, Şifreleme Yöntemi, Tip

ParametreAnlamları
1. Parametre = DosyaGönderilecek dosya.
2. Parametre = İsimDosyanın görünmesi istenilen adı.
3. Parametre = [Şifreleme Yöntemi = 'base64']Şifreleme yöntemi.
4. Parametre = [Tip = 'application/octet-stream']Gönderim tipi.

import::library('Email');

$ayarlar = array(
	'host' => 'mail.zntr.net',
    	'port' => 587,
    	'username' => 'bilgi@zntr.net',
    	'password' => 'zntr1234'
);

email::open();
email::settings($ayarlar);
email::sender('bilgi@zntr.net','zntr');
email::receiver('iletisim@zntr.net','zntr');
email::subject('Ornek Mail Gönderimi');
email::message('Merhaba ben zntr örnek bir mail gönderdim.');
email::add_attachment('resim.png','resim');
email::send();
email::close();

E-posta İşlemleri Sırasında Hata Bilgisi Almak email::error();

email::error() , email::detail()

E-posta gönderimi sırasında oluşan hatanın kaynağını öğrenmek için kullanılır. E-posta gönderimi sonlandırılmadan ve send() yönteminden hemen sonra kullanılmalıdır.

import::library('Email');

$ayarlar = array(
	'host' => 'mail.zntr.net',
    	'port' => 587,
    	'username' => 'bilgi@zntr.net',
    	'password' => 'zntr1234'
);

email::open();
email::settings($ayarlar);
email::sender('bilgi@zntr.net','zntr');
email::receiver('iletisim@zntr.net','zntr');
email::subject('Ornek Mail Gönderimi');
email::message('Merhaba ben zntr örnek bir mail gönderdim.');
email::add_attachment('resim.png','resim');
email::send();

echo email::error();  // Herhangi bir hata oluşursa çıktısını verecektir aksi halde false değeri dönecektir.

var_dump(email::detail(););  // E-postanın detayları hakkında bir dizi bilgi verecektir.

email::close();
