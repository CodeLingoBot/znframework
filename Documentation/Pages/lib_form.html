Form Kütüphanesini Dahil Etmek

import::library('Form');

Form Açmak

form::open( string $isim , [ array $ozellikler ] )

Form tagı açmak için kullanılır. 2 parametresi vardır. Form İsmi, Form Özellikleri

ParametreAnlamları
1. Parametre = Form İsmiForm nesnesinin ismi.
2. Parametre = [Form Özellikleri]Form nesnesinin özellik ve değerleri.

echo form::open('OrnekForm', array('method' => 'post')); // <form name="OrnekForm" id="OrnekForm" method="post">

Form Kapatmak

form::close()

Form tagını kapatmak için kullanılır.

    echo form::open('OrnekForm', array('method' => 'post'));  // <form name="OrnekForm" id="OrnekForm" method="post">
    echo form::close();  // </form>

Yazı Kutusu Oluşturmak

form::text( string $isim , string $deger , [ array $ozellikler ] )

Yazı kutusu oluşturmak için kullanılır. 3 parametresi vardır. Nesne İsmi, Nesne Değeri, Nesne Özellikleri

ParametreAnlamları
1. Parametre = Nesne İsmiText nesnesinin ismi.
2. Parametre = Nesne DeğeriText nesnesinin üzerinde görünecek yazı.
3. Parametre = [Nesne Özellikleri]Text nesnesinin özellik ve değerleri.

echo form::text('YaziKutusu', 'ZN Kod Çatısına Hoş Geldiniz!', array('maxlength' => 10'));
// <input type="text" name="YaziKutusu" id="YaziKutusu" value="ZN Kod Çatısına Hoş Geldiniz!" maxlength="10">

ÖNEMLİ
Özellik olarak id değeri verilmediği sürece form nesnelerinin isim yani name değeri neyse id değeride o olacaktır.

Seçim Listesi Oluşturmak

form::select( string $isim , array $secenekler , string/int $secili_deger , [ array $ozellikler ] )

Seçim listesi oluşturmak için kullanılır. 4 parametresi vardır. Nesne İsmi, Seçenekler, Seçili Olan Seçenek, Nesne Özellikleri

,
ParametreAnlamları
1. Parametre = Nesne İsmiSelect nesnesinin ismi.
2. Parametre = SeçeneklerSelect nesnesine eklenecek seçenekler.
3. Parametre = Seçili DeğerSelect nesnesinde hangi seçeneğin seçili olacağı.
4. Parametre = [Nesne Özellikleri]Select nesnesinin özellik ve değerleri. 

$secenekler = array(
    '34' => 'İstanbul',
    '19' => 'Çorum'
);
echo form::select('sehir', $secenekler, '19');
 
 // <select name="sehir">
    	<option value="19" selected="selected">Çorum</option>
        <option value="34">İstanbul</option>
</select>
