FTP Bağlantısı Oluşturmak

ftp::connect( [ array $ayarlar ] )

Config/Ftp.php dosyasında aşağıdaki FTP ayarları yer almaktadır. Bu ayarlar düzgün yapılmış ise ftp::connect() yönteminin kullanımına gerek yoktur.

$config['Ftp']['host'] = '';
$config['Ftp']['user'] = '';
$config['Ftp']['password'] = '';
$config['Ftp']['timeout'] = 90;
$config['Ftp']['port'] = 21;
$config['Ftp']['ssl_connect'] = false;

ParametreAnlamları
[ array Ayarlar ]Oluşturulacak dizin adını ifade eder.
Ayarlar Dizisinin İçereceği Geçerli Ayar Parametreleri
hostBağlantı sağlanacak FTP adresi. Örnek: ftp.ornek.com
userFTP adresinin kullanıcı adı. Örnek: ornekcom
passwordKullanıcı şifresi. Örnek: x34wdfw1
[ port = 21]FTP port numarası varsayılan olarak 21 ayarlıdır.
[ timeout = 90]FTP bağlantısı zaman aşımı süresi var sayılan olarak 90 saniye ayarlıdır.
[ ssl_connect = false]FTP ssl bağlantısı sağlamak için kullanılır. Varsayılan olarak false ayarlıdır.

$ayarlar = array
(
    'host' => 'ftp.ornek',
    'user' => 'ornekcom',
    'password' => 'orneksifre'
);
        
ftp::connect($ayarlar);  // Bağlantı oluşturuluyor.

Dosya Boyutunu Öğrenmek

ftp::file_size( string $dosya_adi )

Sunucudaki dosya veya bir dizin içindeki dosyaların toplam boyutunu öğrenmek için kullanılır.

ParametreAnlamları
string Dosya AdıBoyutu öğrenilecek dosya adını ifade eder.
[string Boyut Türü = b]Boyutun hangi türde gösterileceği belirlenir.
Boyut Türü Parametresinin Alabileceği Değerler
bBoyutu byte cinsinden belirtir.
kbBoyutu kilo byte cinsinden belirtir.
mbBoyutu mega byte cinsinden belirtir.
gbBoyutu giga byte cinsinden belirtir.

ftp::file_size('x/OrnekDosya.php'); // OrnekDosya.php dosyasının boyutunu verir.
ftp::file_size('x', 'mb'); // x dizini içerisindeki tüm dosyaların toplam boyutunu mega byte cinsinden verir.

Sunucudan Dosya İndirmek

ftp::download( string $sunucudan_yuklenecek_dosya , string $yerele_indirilecek_yol , [ string $yukleme_modu = 'ascii' ] )

Yerelden sunucuya dosya yüklemek için kullanılır.

ParametreAnlamları
string Dosya AdıSunucudan yüklenecek dosya adı.
string Yerel YolDosyanın yerelde hangi yola yükleneceğini belirten yol.
[string Yükleme Modu = 'ascii']Dosyanın yükleme modu.
Yükleme Modu Parametresinin Alabileceği Değerler
asciiFTP_ASCII
binaryFTP_BINARY.

ftp::download('sunucu/OrnekDosya.php' , ''yerel/dosyalar'');
ftp::download('sunucu/OrnekDosya.php' , ''yerel/dosyalar'' , 'binary');

Dizin İçindeki İstenilen Dosya ve Dizinleri Öğrenmek

ftp::files( string $hedef_dizin , [ string $dosya_uzantisi ] )

Dizin içerisindeki istenilen tüm dosya ve dizinlerlerin listesini almak için kullanılır. 2 parametresi vardır. Hedef Dizin, Uzantı

ParametreAnlamları
1. Parametre = Hedef Dizinİçindekileri listelenecek dizin adı.
2. Parametre = [İsteğe Bağlı - Uzantı]Listede görüntülenmesi istenen uzantılı dosyalar. Hiç bir parametrik değer girilmez ise tüm dosya ve dizinleri listeleyecektir.

var_dump(ftp::files('x/DizinVeIcindekiler')); // x/DizinVeIcindekiler dizini içerisindeki dosya ve dizinleri listeleyecektir.

html uzantılı dosyaları listeyelim.

var_dump(ftp::files('x/DizinVeIcindekiler','html')); // x/DizinVeIcindekiler dizini içerisindeki html uzantılı dosyaları listeleyecektir.

Sadece dizinleri listeyelim.

var_dump(ftp::files('x/DizinVeIcindekiler','dir')); // x/DizinVeIcindekiler dizini içerisindeki sadece dizinleri listeleyecektir.

Dosya ve Dizin Yetkilerini Düzenlemek

ftp::permission( string $dosya_adi , [ numeric $yetki_numarasi = 0755 ] )

Sunucu üzerinde dosyayı silmek için kullanılır.

ParametreAnlamları
string Dosya AdıYetkileri düzenlenecek dosya adını ifade eder.
[numeric Yetki Kodu]Yetki kodu.

ftp::permission('x/OrnekDosya.php', 0777);

Dosya ve Dizin İşlemlerinde Hataları Denetlemek

ftp::error()

Sunucu işlemleri esnasında oluşan hataları öğrenmek için kullanılır. Hata oluşmussa hatanın içeriğini yoksa false dönecektir.

ftp::error(); // Silinecek dizin bulunamadı!
