Creator Aracını Dahil Etmek

import::tool('Creator')

Şifre Oluşturma Aracı

password_creator( [ numeric $uzunluk = 6 ] , [ string $karakter_tipi = 'all' ] )

Rasgele şifre üretmek için kullanılır.

NoParametrelerAnlamları
1[ string/int Uzunluk = "6" ]Şifrenin karakter uzunluğu. Varsayılan: 6
2[ string Karakter Tipi = "all" ]Şifreleme için kullanılacak karakter tipi. Varsayılan: all
NoKarakter Tipleri
1all 0-9, A-Z
2numeric 0-9
3string A-Z

echo password_creator(); /* Çıktı: lF6jye */
echo password_creator('10', 'string'); /* Çıktı: BOPTstMAXA */
echo password_creator('10', 'numeric'); /* Çıktı: 738693701*/
