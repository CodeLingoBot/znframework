import::tool('Parser')

Tür Dönüştürme Aracı Aracı

parser( variable $degisken , [ string $tip = 'int' ] )

Değikenlerde tür dönüştürmek için kullanılır.

NoParametrelerAnlamları
1variable Türü Deiştirilecek DeğişkenTür dönüşümü yapılcak değişken.
2[ string Tip = 'int' ]Hangi türe dönüştürüleceği.
Dönüştürülebilecek Türler
1int veya integer
2bool veya boolean
3string
4float
5double
6real
7object
8array
9unset

$metin = 'zntr';
echo parser($metin , 'int'); /* Çıktı: 0 */
echo parser($metin , 'bool'); /* Çıktı: 1 */
var_dump( parser($metin , 'array') ); /* Çıktı: array (size=1) 0 => string 'ozan' (length=4) */
